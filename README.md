- サーバー　サービスを提供するコンピューター、なにを提供するかによって呼び方が変わる
- クライアント　サービスの提供を受ける
- プロトコル　コンピュータ同士の通信の取り決め
- WEBサーバーの役割 HTTPのリクエストを受け、リクエストに応じたHTTPレスポンスを返すこと
- Webサイトにアクセスする際にURLを入力
- URLを入力すると、ブラウザからサーバにHTTPリクエストが送信される
- リクエストを受け取ったサーバは、ブラウザにHTTPレスポンスを返す
- そしてブラウザは受け取ったレスポンスを解析して画面を描画している
- 環境とはサーバーのこと
- 仮想環境の構築に使用される代表的なソフトウェアとして Dockerがある
